Introdução

Quando tags como font e atributos de cores foram adicionados à especificação do HTML 3.2, começou um pesadelo para os desenvolvedores da web. O desenvolvimento de grandes sites, onde fontes e informações de cores foram adicionadas a cada página, tornou-se um processo longo e caro.

Para resolver este problema, o World Wide Web Consortium (W3C) criou CSS. CSS removeu a formatação de estilo da página HTML!

  • CSS descreve como os elementos HTML devem ser exibidos na tela, no papel ou em outras mídias.
  • CSS economiza muito trabalho . Pode controlar o layout de várias páginas da web de uma só vez.
    • Sintaxe

    Um conjunto de regras CSS consiste em um seletor e um bloco de declaração:

  • O seletor aponta para o elemento HTML que você deseja estilizar.
  • O bloco de declaração contém uma ou mais declarações separadas por ponto e vírgula.
  • SUma declaração CSS sempre termina com um ponto-e-vírgula e os blocos de declaração são cercados por chaves.
    • Background Color

    A background-colorpropriedade especifica a cor de fundo de um elemento. A cor de fundo de uma página é configurada assim:

    body { background-color: lightblue; }

    A background-image propriedade especifica uma imagem para usar como plano de fundo de um elemento. Por padrão, a imagem é repetida para cobrir todo o elemento. A imagem de fundo de uma página pode ser configurada assim:

    body { background-image: url("paper.gif"); }

    Por padrão, a background-imagepropriedade repete uma imagem na horizontal e na vertical. Algumas imagens devem ser repetidas apenas horizontalmente ou verticalmente, ou parecerão estranhas, assim:

    body { background-image: url("gradient_bg.png"); }
    CSS Borders

    A border-stylepropriedade especifica o tipo de borda a ser exibido.Os seguintes valores são permitidos:

  • dotted -Define uma borda pontilhada
  • dashed - Define uma borda tracejada
  • solid - Define uma borda sólida
  • double - Define uma fronteira dupla
  • groove- Define uma borda com sulcos 3D. O efeito depende do valor da cor da borda
  • ridge- Define uma borda 3D sulcada. O efeito depende do valor da cor da borda
  • inset- Define uma borda inserida em 3D. O efeito depende do valor da cor da borda
  • outset- Define uma borda de partida 3D. O efeito depende do valor da cor da borda
  • none - não define fronteiras
  • hidden - Define uma borda oculta

    • A border-widthpropriedade especifica a largura das quatro bordas. A largura pode ser definida como um tamanho específico (em px, pt, cm, em, etc) ou usando um dos três valores predefinidos: thin, medium ou thick. A border-widthpropriedade pode ter de um a quatro valores (para a borda superior, borda direita, borda inferior e borda esquerda).

    p.one { border-style: solid; border-width: 5px; } p.two { border-style: solid; border-width: medium; } p.three { border-style: solid; border-width: 2px 10px 4px 20px; }
    CSS Margins

    As marginpropriedades CSS são usadas para criar espaço em torno de elementos, fora de qualquer borda definida. Com CSS, você tem controle total sobre as margens. Existem propriedades para definir a margem para cada lado de um elemento (superior, direita, inferior e esquerda).

    O exemplo a seguir define margens diferentes para todos os quatro lados de um elemento:

    p { margin-top: 100px; margin-bottom: 100px; margin-right: 150px; margin-left: 80px; }

    CSS tem propriedades para especificar a margem para cada lado de um elemento:

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left

    • Você pode definir a propriedade de margem autopara centralizar horizontalmente o elemento dentro de seu contêiner. O elemento ocupará a largura especificada e o espaço restante será dividido igualmente entre as margens esquerda e direita:

    div { width: 300px; margin: auto; border: 1px solid red; }
    CSS Padding

    As padding propriedades CSS são usadas para gerar espaço em torno do conteúdo de um elemento, dentro de qualquer borda definida. Com CSS, você tem controle total sobre o preenchimento. Existem propriedades para definir o preenchimento de cada lado de um elemento (superior, direita, inferior e esquerda).

    Exemplo:

    div { padding-top: 50px; padding-right: 30px; padding-bottom: 50px; padding-left: 80px; }

    A widthpropriedade CSS especifica a largura da área de conteúdo do elemento. A área de conteúdo é a parte dentro do preenchimento, borda e margem de um elemento ( o modelo de caixa ). Portanto, se um elemento tiver uma largura especificada, o preenchimento adicionado a esse elemento será adicionado à largura total do elemento. Isso geralmente é um resultado indesejável.

    Exemplo:

    div { width: 300px; padding: 25px; }
    CSS Height e Width

    As propriedades heighte widthsão usadas para definir a altura e a largura de um elemento. O heighte widthpode ser definido como automático (isso é o padrão. Significa que o navegador calcula a altura e a largura) ou pode ser especificado em valores de comprimento , como px, cm, etc., ou em porcentagem (%) do bloco de contenção.

    Exemplo:

    div { height: 200px; width: 50%; background-color: powderblue; } div { height: 100px; width: 500px; background-color: powderblue; }

    As propriedades heighte widthnão incluem preenchimento, bordas ou margens; eles definem a altura / largura da área dentro do preenchimento, borda e margem do elemento!

    CSS Box Model

    Todos os elementos HTML podem ser considerados como caixas. Em CSS, o termo "modelo de caixa" é usado quando se fala em design e layout. O modelo de caixa CSS é essencialmente uma caixa que envolve todos os elementos HTML. Consiste em: margens, bordas, preenchimento e o conteúdo real.

    O modelo de caixa nos permite adicionar uma borda ao redor dos elementos e definir o espaço entre os elementos.

    Exemplo:

    div { width: 300px; border: 25px solid green; padding: 25px; margin: 25px; }

    Para definir a largura e a altura de um elemento corretamente em todos os navegadores, você precisa saber como o modelo de caixa funciona.

    div { width: 320px; padding: 10px; border: 5px solid gray; margin: 0; }

    A largura total de um elemento deve ser calculada da seguinte forma:

    Largura total do elemento = largura + preenchimento à esquerda + preenchimento à direita + borda esquerda + borda direita + margem esquerda + margem direita

    A altura total de um elemento deve ser calculada da seguinte forma: Altura total do elemento = altura + preenchimento superior + preenchimento inferior + borda superior + borda inferior + margem superior + margem inferior

    CSS Outline

    A outline-stylepropriedade especifica o estilo do contorno e pode ter um dos seguintes valores:

  • dotted - Define um contorno pontilhado
  • dashed - Define um contorno tracejado
  • solid - Define um contorno sólido
  • double - Define um contorno duplo
  • groove - Define um contorno sulcado em 3D
  • ridge - Define um contorno 3D
  • inset - Define um contorno em 3D
  • outset - Define um esboço inicial em 3D
  • none - não define nenhum esboço
  • hidden - Define um contorno oculto

    • Nenhuma das outras propriedades de contorno terá qualquer efeito, a menos que a outline-stylepropriedade esteja definida!

    A outline-colorpropriedade é usada para definir a cor do contorno:

    p.ex1 { border: 1px solid black; outline-style: solid; outline-color: red; } p.ex2 { border: 1px solid black; outline-style: double; outline-color: green; } p.ex3 { border: 1px solid black; outline-style: outset; outline-color: yellow; }
    CSS Text

    A colorpropriedade é usada para definir a cor do texto.

    A cor de texto padrão de uma página é definida no seletor de corpo.

    body { color: blue; } h1 { color: green; }

    A text-alignpropriedade é usada para definir o alinhamento horizontal de um texto. Um texto pode ser alinhado à esquerda ou à direita, centrado ou justificado. O exemplo a seguir mostra alinhado ao centro e texto alinhado à esquerda e à direita (o alinhamento à esquerda é padrão se a direção do texto for da esquerda para a direita e o alinhamento à direita for padrão se a direção do texto for da direita para a esquerda):

    h1 { text-align: center; } h2 { text-align: left; } h3 { text-align: right; }
    CSS Icons

    A maneira mais simples de adicionar um ícone à sua página HTML é com uma biblioteca de ícones, como o Font Awesome.

    Adicione o nome da classe de ícone especificada a qualquer elemento HTML.

    CSS Links
    Os links podem ter um estilo diferente, dependendo do estado em que estão.

    Os quatro estados de links são:

  • a:link - um link normal não visitado
  • a:visited - um link que o usuário visitou
  • a:hover - um link quando o usuário passar por cima dele
  • a:active - um link no momento em que é clicado

    • Exemplo:

    /* unvisited link */ a:link { color: red; } /* visited link */ a:visited { color: green; } /* mouse over link */ a:hover { color: hotpink; } /* selected link */ a:active { color: blue; }

    A text-decorationpropriedade é usada principalmente para remover os sublinhados dos links:

    Exemplo:

    a:link { text-decoration: none; } a:visited { text-decoration: none; } a:hover { text-decoration: underline; } a:active { text-decoration: underline; }
    CSS Lists

    A list-style-typepropriedade especifica o tipo de marcador de item da lista. O exemplo a seguir mostra alguns dos marcadores de itens de lista disponíveis:

    l.a { list-style-type: circle; } ul.b { list-style-type: square; } ol.c { list-style-type: upper-roman; } ol.d { list-style-type: lower-alpha; }
    CSS Opacity

    A opacitypropriedade pode ter um valor de 0,0 a 1,0. Quanto menor o valor, mais transparente:

    Ezemplo:

    img { opacity: 0.5; filter: alpha(opacity=50); /* For IE8 and earlier */ }

    A opacity propriedade é frequentemente usada junto com o :hover seletor para alterar a opacidade ao passar o mouse:

    img { opacity: 0.5; filter: alpha(opacity=50); /* For IE8 and earlier */ } img:hover { opacity: 1.0; filter: alpha(opacity=100); /* For IE8 and earlier */ }
    Float e Clear

    A floatpropriedade é usada para posicionar e formatar conteúdo, por exemplo, deixar uma imagem flutuar para o texto em um contêiner.

    A floatpropriedade pode ter um dos seguintes valores:

  • left - O elemento flutua à esquerda do contêiner
  • right- O elemento flutua à direita do seu contêiner
  • none - O elemento não flutua (será exibido exatamente onde ocorre no texto). Este é o padrão
  • herit - O elemento herda o valor float de seu pai

    • Em seu uso mais simples, a floatpropriedade pode ser usada para envolver o texto em torno das imagens.

    O exemplo a seguir especifica que uma imagem deve flutuar para a direita em um texto:

    img { float: right; }
    CSS Tables

    A aparência de uma tabela HTML pode ser muito melhorada com CSS.

    Para especificar bordas de tabela em CSS, use a borderpropriedade. O exemplo abaixo especifica uma borda preta. table, th, td { border: 1px solid black; }

    A border-collapsepropriedade define se as bordas da tabela devem ser recolhidas em uma única borda:

    table { border-collapse: collapse; } table, th, td { border: 1px solid black; }

    O exemplo a seguir alinha o texto à esquerda:

    th { text-align: left; }
    CSS Overflow

    A overflowpropriedade especifica se é necessário recortar o conteúdo ou adicionar barras de rolagem quando o conteúdo de um elemento é grande demais para caber em uma área especificada. A overflowpropriedade possui os seguintes valores:

  • visible- Padrão. O estouro não é cortado. Ele renderiza fora da caixa do elemento
  • hidden - O estouro é cortado e o restante do conteúdo ficará invisível
  • scroll - O estouro é recortado, mas uma barra de rolagem é adicionada para ver o restante do conteúdo
  • auto - Se o estouro for cortado, uma barra de rolagem deve ser adicionada para ver o restante do conteúdo

    • Nota: A overflowpropriedade só funciona para elementos de bloco com uma altura especificada. Nota: No OS X Lion (no Mac), as barras de rolagem ficam ocultas por padrão e são mostradas apenas quando são usadas (embora "estouro: rolagem" esteja definido).

    Por padrão, o estouro é visible, o que significa que não é cortado e é renderizado fora da caixa do elemento:

    Você pode usar a propriedade de overflow quando quiser ter um melhor controle do layout. A propriedade de overflow especifica o que acontece se o conteúdo ultrapassar a caixa de um elemento.

    div { width: 200px; height: 50px; background-color: #eee; overflow: visible; }
